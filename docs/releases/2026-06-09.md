# NXP Monkey Release 2026-06-09

## [2026.6.9]

Release `2026.6.9` improves part discovery and fetch ergonomics for masked
NXP portfolio keys.

Highlights:

- Resolve concrete orderable MPNs such as `MIMX9352CVVXMAB` to masked
  portfolio keys such as `MIMX9352xxxxM`.
- Allow positional `fetch` inputs such as `MIMX93` to expand to matching
  portfolio keys before downloads begin.
- Fall back from `search` to the NXP portfolio map when the local SQLite index
  is absent, while keeping indexed search as the fast offline path.
- Return a clearer fetch error when a selection matches but no SDK variant can
  be fetched.
- Keep cache layout, JSON sidecars, XML-to-JSON mirrors, and public command
  names unchanged.
