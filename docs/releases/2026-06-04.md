# NXP Monkey Release 2026-06-04

## [2026.6.4]

Release `2026.6.4` aligns NXP Monkey with the current Wavenumber Python package
standard.

Highlights:

- Add explicit repository hygiene files, setup docs, architecture docs, release
  notes, and documented standards exceptions.
- Add `nxp-monkey version` and `nxpm version` for package/runtime version
  reporting.
- Move CI and release workflows to native Node 24 GitHub Actions.
- Move CI and release signoff to Rack across Ubuntu, Windows, and macOS.
- Update the release/signoff dependency baseline to `wn-rack>=2026.6.4`.
- Add a Python signoff ratchet for current complexity, function-size,
  annotation, and `Any` baselines.
- Keep NXP KEX fetch, index, search, cache, XML, and JSON behavior unchanged.
