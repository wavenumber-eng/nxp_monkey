nxp-monkey help

design / cli / help

Purpose

Convenience alias: nxp-monkey help <cmd> is equivalent to nxp-monkey <cmd> --help. With no topic, prints the top-level help.

Usage

nxp-monkey help
nxp-monkey help COMMAND

Output

Whatever the targeted subparser's --help prints. No other side effects.
